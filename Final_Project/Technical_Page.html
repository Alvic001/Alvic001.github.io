Github

Github is like a big playground for coding projects. It's a website where people can store and share their code with others. Think of it as a place where you can keep track of changes you make to your code, collaborate with others on projects, and even showcase your work to the world. It's super handy for teamwork and making sure everyone's on the same page when working on a project together.

Github sample commands *

git init
git add .
git commit -m "Initial commit"
git remote add origin your_github_repository_url.git
git push -u origin master

VS code

VS code is like a digital workshop for writing code. It's a program you can use on your computer to write and edit code for all sorts of projects. Think of it as a super helpful tool that makes coding easier with features like syntax highlighting (color-coding your code to make it easier to read), code suggestions, and debugging tools. Plus, you can add extensions to customize it for different programming languages and tasks. It's great for both beginners and experienced coders alike!

VS code sample shortcuts: *

Ctrl + F: Find text in the current file.
Ctrl + H: Replace text in the current file.
Ctrl + D: Select the next occurrence of the current word.
Ctrl + Shift + F: Search across all files in the project.
Ctrl + P: Quick open a file by name.
Ctrl + Shift + N: Open a new window.
Ctrl + Shift + P: Open the command palette for access to all commands.
Ctrl + Shift + L: Select all occurrences of the current selection.
Ctrl + / (or Ctrl + Shift + 7): Toggle line comment.
HTML

HTML is like the skeleton of a webpage. It's a simple language that web browsers understand. With HTML, you can structure your webpage by adding different elements like headings, paragraphs, images, and links. Think of it as the blueprint that tells your browser how to display content on a webpage. It's the basic building block for creating websites on the internet.

HTML code example: *

-!DOCTYPE - html declares the document type and version of HTML being used.
- html - is the root element of the HTML document.
- head - contains meta-information about the HTML document, like the title.
- title - sets the title of the webpage displayed in the browser tab.
- body - contains the visible content of the webpage.
- h1 - is a heading element.
- p - is a paragraph element.
- img - embeds an image into the webpage, with the src attribute specifying the image URL
CSS

CSS is like the stylist of a webpage. It stands for Cascading Style Sheets. With CSS, you can make your webpage look pretty by adding colors, fonts, spacing, and more. Think of it as the decorator that enhances the appearance of your HTML elements. It helps you control how your webpage looks and feels to the users who visit it.

example: *

With CSS, you can make your webpages look really cool! You can change colors, fonts, sizes, and even the layout of your webpage. Imagine you have a plain webpage with just text and pictures. CSS lets you style the text with different colors and fonts, make the pictures bigger or smaller, and arrange everything in a neat way. It's like giving your webpage a makeover to make it stand out and look awesome!
WSL

WSL is like having a secret passage on your Windows computer that leads to the world of Linux. It lets you run Linux programs and use Linux commands directly within Windows, almost like having two operating systems in one computer. With WSL, you can do things like run Linux software, use terminal commands, and even develop software using tools that are typically found in Linux environments, all without having to switch between different operating systems. It's like having the best of both worlds!
